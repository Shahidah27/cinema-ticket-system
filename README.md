# 🎬 Cinema Ticket System

A professional desktop application built for CSC301 coursework using Visual Studio. The system features distinct interfaces for customers and administrators, providing a seamless workflow for movie browsing, food selection, seat assignment, and transactional data processing.

---

## 🚀 Key Features

### 🔐 Multi-Role Authentication
* Features split login credentials for both **Staff/Admin** and **Customers** to access their respective system dashboards securely.

### 👤 Customer Booking Engine
* **Dynamic Movie Selection:** Browse comprehensive movie details, descriptions, genres, and showtimes directly pulled from the database.
* **Seat Category Pricing:** Select standard (RM 15) or premium (RM 25) seat classes with custom seat number inputs.
* **Integrated Food & Beverage Module:** Order specific snacks, beverages, or combo meals with real-time total price accumulation.
* **Receipt Generation:** Generates a structured invoice breakdown including personal data, itemized pricing, and payment gateway selection (Online Banking, DuitNow QR, Apple Pay, etc.) with print preview support.

### ⚙️ Administrator Dashboard (CRUD Management)
* **Movie Information Logs:** Complete control panel allowing admins to **Add, Save, Update, and Delete** system movies and descriptions, fully integrated with a live data grid view.
* **Seat Information Logs:** Allows administrative tracking of seat IDs, hall numbers, and active seat availability status.

---

## 🛠️ Tech Stack & Tools

* **Frontend/UI:** VB.NET (Windows Forms)
* **IDE:** Visual Studio Community
* **Database:** Microsoft Access (`.accdb`)
* **Data Connectivity:** ADO.NET DataSets & TableAdapters
